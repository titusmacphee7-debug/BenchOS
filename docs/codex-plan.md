# BenchOS Codex Plan

## Phase 1

Build the static React + Vite + TypeScript app shell with mock data and polished dark UI.

Done when:
- Every route renders.
- The app visually matches the BenchOS dark mockups.
- `npm run build` passes.

## Phase 2

Add local-first data:
- Dexie IndexedDB.
- Tool library seed data.
- Tool search.
- ToolPicker flows.
- My Tools CRUD.
- Materials CRUD.

## Phase 3

Add the intelligence loop:
- Projects CRUD.
- Project requirements.
- Readiness engine.
- Wishlist conversion.
- Real dashboard data.

## Phase 4

Add workshop activity:
- Tool usage logs.
- Material usage logs.
- Maintenance logs.
- BenchXP events.
- Tool Mastery progress.

## Phase 5

Polish for beta:
- Import/export.
- Onboarding.
- Empty states.
- Tests.
- PWA setup.
- Packaging path.
