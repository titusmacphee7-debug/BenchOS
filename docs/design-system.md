# BenchOS Design System

BenchOS uses a dark workshop command-center interface.

## Tokens

- Background: `#07090D`
- Surface: `#10141A`
- Raised surface: `#151A22`
- Border: `#252B36`
- Text: `#F5F7FA`
- Muted text: `#8B93A1`
- Accent orange: `#FF6A00`
- Soft orange: `#3A1F0D`
- Green: `#22C55E`
- Yellow: `#F59E0B`
- Red: `#EF4444`
- Blue: `#3B82F6`
- Purple: `#8B5CF6`

## UI Rules

- Use dense layouts for workshop data.
- Use cards for individual data regions and repeated items.
- Use tables for inventory-style lists.
- Use right-side detail panels for selected tools, materials, and projects.
- Use orange for primary actions and active navigation.
- Use status pills for condition, readiness, stock, and priority.
- Use lucide-react icons.
- Do not use copyrighted tool product photos.
