# Example Style Notes

The attached circular saw reference is only broad inspiration for a simple workshop-icon direction. Do not copy the composition, proportions, blade placement, handle shape, line quirks, or any visible artifact from it.

Reusable style cues:
- Black line art.
- Monoline technical illustration.
- Bold outside silhouette.
- Sparse interior contour lines.
- Hand-drawn but tidy.
- Centered single tool.
- Square icon-friendly crop.

Do not include:
- Text.
- Labels.
- Watermarks.
- Signatures.
- Brand marks.
- Brand colors.
- Photorealistic lighting.
- Shop background scenes.
